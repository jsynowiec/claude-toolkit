# claude-toolkit

A curated collection of plugins that extend Claude Code with specialized agents, skills, and commands.

## Plugins

| Plugin | Description |
|--------|-------------|
| [stackshift](./plugins/stackshift) | Modernize legacy codebases, upgrade runtimes, migrate frameworks, and reduce technical debt |
| [book-forge](./plugins/book-forge) | Build professionally structured ePub 3 e-books from Markdown sources using pandoc |
